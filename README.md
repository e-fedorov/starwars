# starwars
Training application - requesting https://swapi.dev/ API and displays received data on screen
Email - fedorov-e@keysystems.ru

Домашнее задание - дописать приложение, по нажатию на кнопку показать должна открываться новый фрагмент или activity со списком резидентов планеты, 
список резидентов берется из api, по шаблону https://swapi.dev/api/planets/<id планеты>/?format=json (оттуда берем массив residents, в котором отображаются 
список ссылок для получения информации о конкретном резиденте, например https://swapi.dev/api/people/1/?format=json, берем оттуда name). Чтобы получить данные
о всех жителях надо сделать запрос для каждого url из списка, затем отобразить в новой активити.
Домашнюю работу сбросить на почту fedorov-e@keysystems.ru
